RandomSeats
===========
수정 예정
1. 화면 돌릴 떄 오류
2. 공유기능, 사진 저장기능
3. 기록 저장기능
4. 번호 보일 떄 애니메이션 기능, 효과음
5. 가로모드에서 프레그먼트로 레이아웃 바꾸기
6. 메뉴 만들기 (프리미엄 버전 구매)
7. 프리미엄 버전 만들기 (광고 제거, 번호 표시할 때 다양한 애니메이션, 다양한 테마)
8. 자리 세팅 화면에서 초기화 버튼 만들기
